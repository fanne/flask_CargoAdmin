# flask_CargoAdmin

一个flask小项目，后台用了flaskadmin，前端带有全文索引的功能。

其中索引采用了[Flask-WhooshAlchemyPlus](https://github.com/Revolution1/Flask-WhooshAlchemyPlus)这个功能模块。
